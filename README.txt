 Summer Internship – Cybersecurity Learning Journey

Overview

During my summer internship, I focused on building foundational knowledge in cybersecurity through hands-on practice, self-learning, and practical exercises. The internship helped me gain experience with Linux-based environments, cybersecurity tools, reconnaissance techniques, and security-focused learning platforms.

 Key Learning Areas

 Kali Linux Fundamentals

* Learned and practiced essential Kali Linux commands for system navigation, package management, networking, and user management.
* Explored cybersecurity tools available in Kali Linux, including:

  * Nmap
  * Dig
  * Whois
  * Tcpdump
  * John the Ripper
* Gained understanding of networking concepts such as ports, services, DNS records, and packet analysis.

 TryHackMe Labs

* Completed beginner-friendly TryHackMe learning paths and labs.
* Practiced Linux fundamentals, networking concepts, reconnaissance techniques, and security methodologies.
* Improved problem-solving skills through hands-on cybersecurity challenges.

 OverTheWire Bandit

* Successfully worked through multiple Bandit levels.
* Learned Linux command-line usage, file manipulation, permissions, encoding concepts, and basic security principles.
* Developed confidence in navigating Linux environments and solving practical tasks.

 Report Writing and Documentation

* Prepared technical reports documenting learning activities, observations, and outcomes.
* Practiced organizing cybersecurity findings in a structured and professional format.
* Improved technical communication and documentation skills.
